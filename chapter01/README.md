# Bun 시작하기

## MacOS/리눅스에서 설치

```sh
$ curl -fsSL https://bun.sh/install | bash
```

## Homebrew로 설치

```sh
$ brew install oven-sh/bun/bun
```

## 윈도우에서 설치

```sh
$ powershell -c "irm bun.sh/install.ps1|iex"
```

## Scoop으로 설치

```sh
$ scoop install bun
```

## Bun 설치 확인하기

```sh
$ bun -v
```

## Bun 업데이트하기

```sh
$ bun upgrade
```

## hello.js 파일 실행하기

```sh
$ bun hello.js
```

## REPL 실행하기

```sh
$ bun repl
```
